# Elevate for Humanity - Program Curriculum Structure

## Delivery Model

Most course content is delivered through **external partner platforms**. The Elevate LMS serves as:
1. **Student Dashboard** - Track progress across all certifications
2. **JRI SCORM Host** - EmployIndy badges delivered in-platform
3. **Completion Tracking** - Record external certifications
4. **Certificate Issuance** - Issue Elevate completion certificates

## Partner Platforms

| Partner | Certifications | Delivery |
|---------|---------------|----------|
| **EmployIndy** | JRI Badges (6) | SCORM in LMS |
| **CareerSafe** | OSHA 10/30, Bloodborne Pathogens | careersafeonline.com |
| **HSI** | CPR, AED, First Aid | In-person or RSV |
| **Milady/Cengage** | Barber Theory, RISE Safety | External (code: efhcti-rise295) |
| **NRF Foundation** | RISE Up Customer Service/Retail | External |
| **Certiport** | Microsoft, Adobe, QuickBooks, IT | Testing Center |
| **IRS** | VITA, Link & Learn Taxes | irs.gov |
| **National Drug Screening** | DOT Collector | MyDrugTestTraining.com |

## Program Structure

Each program consists of:
1. **JRI Foundation** (8 hours) - Soft skills, delivered via SCORM
2. **Safety Certifications** - OSHA, CPR, etc. via partners
3. **Technical Training** - Industry-specific via partners
4. **Elevate Completion** - Certificate issued upon all requirements met

## Tracking External Completions

Students upload certificates/completion records from partner sites.
Staff verify and mark complete in admin portal.
System tracks:
- Certification name
- Provider
- Completion date
- Expiration date (if applicable)
- Certificate/credential number

## Files

- `program-curriculum-map.json` - Complete mapping of programs to certifications
- `/jri/` - JRI SCORM packages (delivered in LMS)
- Other content delivered externally
