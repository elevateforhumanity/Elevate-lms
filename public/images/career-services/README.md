# Career Services Images

## Hero Banner Image

**Required:** `hero-banner.jpg`

**Specifications:**
- Dimensions: 1920x1080px (16:9 ratio) or similar
- Format: JPG or PNG
- Max file size: 500KB (optimized for web)
- Subject: Career services, professional development, job search theme

**Source:**
Download from: https://artlist.io/text-to-image-ai/creations/a34be5f8-316c-47ed-925d-c65e14bcba67

**Installation:**
1. Download the image from the Artlist URL
2. Rename to `hero-banner.jpg`
3. Place in this directory: `/public/images/career-services/`
4. The image will automatically appear on all career services subpages

**Usage:**
This hero banner appears on:

- Career Counseling page

- Resume Building page
