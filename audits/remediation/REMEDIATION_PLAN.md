## Content Templates

### For "How It Works" Pages
```
1. Hero with clear value proposition
2. Step-by-step process (3-5 steps)
3. Benefits/features grid
4. FAQ section
5. CTA to next action
```

### For Portal/Dashboard Pages
```
1. Welcome message with user context
2. Quick actions grid
3. Status/progress indicators
4. Recent activity
5. Help/support link
```

### For Form Pages
```
1. Clear purpose statement
2. Progress indicator (if multi-step)
3. Form fields with validation
4. Help text for complex fields
5. Submit CTA with loading state
```
